# GH-6: Add dash log command to view completed work history

## Spec
- New `dash log` command parses history.log and prints a reverse-chronological timeline
- Show IC (closed), RL (release), and CM (commit) entries grouped or chronologically
- Format: date, type label, issue/tag, description — readable at a glance
- No external dependencies; pure shell parsing of the pipe-delimited log
- Complements GH-4 (archive): closed issues will have archived specs, but log shows the timeline
